# SHOP.CO - eCommerce Website

## Overview 
SHOP.CO is a modern eCommerce platform built using React, Zustand, Firebase, and Tailwind CSS. The project is currently in development and aims to provide a seamless shopping experience with features like authentication, product filtering, and cart management.

## Tech Stack
- React - Frontend framework for building the UI.

- Zustand - State management for handling global state.

 - Firebase - Backend for authentication and database services.

- Tailwind CSS - Utility-first CSS framework for styling.

## Features

- User Authentication: Signup and Signin functionality using Firebase Authentication.

 - Product Listing: Browse through a variety of products.

 - Filters & Sorting:

    - Sort by Price: Low to High, High to Low

   - Sort by Ratings

- Additional filters to refine search results

- Shopping Cart: Add and remove products from the cart.

- Responsive UI: Styled with TailwindCSS for a clean and responsive design.

## Tech Stack

- Frontend: React, TailwindCSS

- State Management: Zustand

- Backend & Authentication: Firebase

## Installation

1 - Clone the repository:
```
git clone https://github.com/chaitechie/SHOP.CO.git
```
2 - Navigate to the project directory:
```
cd SHOP.CO
```
3 - Create a env file and add your Firebase credentials:
```
REACT_APP_FIREBASE_API_KEY=your_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_auth_domain
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
```
4 - Start the development server:
```
npm run dev
```
